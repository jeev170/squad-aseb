# Image Upload Guide

Upload your images to this folder via GitHub. The website will automatically display them.

## Required Images:

### Logo
- `squad-logo.png` - Your club logo (recommended: 200x200px, PNG with transparency)

### Hero Banner
- `hero-banner.jpg` - Homepage hero image (recommended: 1920x1080px)

### Mentor
- `mentor.jpg` - Mentor photo (recommended: 400x400px, square)

### Alumni (in `/images/alumni/` folder)

- `alumni-2.jpg` - Priya Sharma
- `alumni-3.jpg` - Arun Patel
- `alumni-4.jpg` - Sneha Reddy
- `alumni-5.jpg` - Vikram Singh
- `alumni-6.jpg` - Anjali Menon

Recommended specs for alumni: 400x400px (square)

### Gallery (in `/images/gallery/` folder)
Upload images named: `gallery-1.jpg`, `gallery-2.jpg`, ... up to `gallery-24.jpg`

### Events (in `/images/events/` folder)
- `event-1.jpg` - Mountain Trek 2025
- `event-2.jpg` - Republic Day Celebration
- `event-3.jpg` - Leadership Workshop
- `event-4.jpg` - Heritage Walk
- `event-5.jpg` - Adventure Camp
- `event-6.jpg` - Independence Day Rally

### Squad Team (in `/images/squad/` folder)
- `president.jpg` - President photo
- `vp-1.jpg` - Vice President 1
- `vp-2.jpg` - Vice President 2
- `treasurer.jpg` - Treasurer

Recommended specs for team photos: 400x400px (square)

## How to Upload via GitHub:
1. Go to your repository on GitHub
2. Navigate to `public/images/`
3. Click "Add file" > "Upload files"
4. Drag and drop your images
5. Commit the changes
